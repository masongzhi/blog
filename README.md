# 个人blog前端（personal blog font-end）
使用create_react_app构建，配合ant-design使用

this appliation install by create-react-app with react-app-rewired (ant-design)

# installation
```bash
# install
npm install

# development
npm start

# build
npm run build

# push docker image
make prod

##### docker deploy
docker run --name blog -d masongzhi/blog:prod
```
